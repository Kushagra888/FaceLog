# 🧠 FaceLog – Facial Recognition Attendance System

**FaceLog** is a Python-based real-time face recognition system that automates attendance tracking using a webcam. It identifies known faces from a preloaded dataset and logs their check-in time to a CSV file.

---

## 📸 Features

- Detects and recognizes faces in real-time via webcam
- Logs attendance with names and timestamps
- Stores attendance records in a `.csv` file
- Lightweight and easy to run locally

---
## Technologies Used
- Python - version 3.10.4
- Flask - version 2.3.2
- OpenCV
- face-recognition 1.3.0
  

## 🚀 Setup

Install the following libraries before running the project:

```bash
pip install opencv-python
pip install -U Flask
pip install face-recognition
```
